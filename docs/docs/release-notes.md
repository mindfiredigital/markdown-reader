# Release Notes

## v1.0.0

Initial public release of Markdown Reader.

### Highlights

- Native desktop Markdown reader
- Cross-platform packaging
- Markdown rendering with GFM support
- Mermaid diagram rendering
- KaTeX math rendering
- Syntax highlighting
- Multiple tabs
- Search
- Themes
- Font zoom
- File watching and live reload
- Recent files
- Markdown file association support

### Downloads

Download the latest builds from GitHub Releases:

[GitHub Releases](https://github.com/mindfiredigital/markdown-reader/releases)

### Known limitations

- Windows and macOS builds may show unsigned app warnings.
- Some Mermaid diagram types may require further visual polish.
- Code signing and notarization are planned for a later release.

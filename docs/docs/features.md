# Features

## Beautiful Markdown rendering

Markdown Reader renders Markdown as a clean readable document, not as an editor preview.

Supported content includes:

- Headings
- Lists
- Task lists
- Tables
- Blockquotes
- Code blocks
- Links
- Images
- Horizontal rules
- Inline HTML after sanitization

## Rich developer content

Markdown Reader supports developer-focused Markdown features:

- Syntax highlighting with Shiki
- Mermaid diagrams
- KaTeX math
- GitHub Flavored Markdown
- Callout-style blocks

## Navigation

- Table of contents sidebar
- Multiple tabs
- Recent files
- In-document search
- Keyboard shortcuts

## Reading controls

- Theme switching
- Font zoom
- Reading width control
- Status bar
- Live reload when the file changes externally

## Desktop integration

- Native file dialogs
- Native installer packages
- Markdown file association support
- Cross-platform builds for Windows, macOS, and Linux

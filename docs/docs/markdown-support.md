# Markdown Support

Markdown Reader focuses on practical Markdown files used in projects, notes, docs, and specifications.

## Supported

- GitHub Flavored Markdown
- Task lists
- Tables
- Code blocks
- Syntax highlighting
- Mermaid diagrams
- KaTeX math
- Links and autolinks
- Local and remote images
- Blockquotes
- Callouts

## Links

External links open in the system browser.

Internal heading links scroll inside the document.

## Images

Local images should be placed relative to the Markdown file.

Example:

```md
![Screenshot](./screenshot.png)
```

Remote images are also supported when the device is online.

```md
![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)
```

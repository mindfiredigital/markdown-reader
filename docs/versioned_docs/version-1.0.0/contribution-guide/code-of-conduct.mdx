# Code of Conduct

Markdown Reader should be a welcoming project for contributors, maintainers, users, and reviewers.

We expect everyone participating in the project to communicate with respect and professionalism. This applies to issues, pull requests, reviews, discussions, documentation, and any other project space.

## Expected behavior

- Be respectful and patient.
- Give constructive feedback.
- Keep discussions focused on the work.
- Assume good intent when possible.
- Help new contributors understand the project.

## Unacceptable behavior

We do not tolerate harassment, insults, personal attacks, trolling, discrimination, or repeated disruptive behavior.

## Maintainer action

Maintainers may remove comments, close discussions, block accounts, or take other reasonable action when someone violates this code of conduct.

## Reporting

If you see behavior that does not follow this code of conduct, contact the maintainers through the project repository or the official project communication channel.

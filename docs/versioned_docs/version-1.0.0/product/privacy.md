# Privacy

Markdown Reader is designed to be offline-first.

## No telemetry

Markdown Reader should not collect analytics, usage data, crash reports, or tracking events.

## Local files stay local

Markdown files are read from your system and rendered on your machine. Normal reading should not require accounts, cloud sync, or background network calls.

## Remote images

If a Markdown file contains remote image URLs, those images may require network access to load. Local images are preferred for fully offline documents.

## Settings storage

User settings such as theme, zoom level, recent files, and reading width should be stored locally in the app configuration directory.

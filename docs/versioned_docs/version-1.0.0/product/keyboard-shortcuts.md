# Keyboard shortcuts

Markdown Reader is designed to be keyboard friendly.

| Shortcut               | Action                         |
| ---------------------- | ------------------------------ |
| `Ctrl/Cmd + O`         | Open file                      |
| `Ctrl/Cmd + Shift + O` | Open folder                    |
| `Ctrl/Cmd + W`         | Close current tab              |
| `Ctrl/Cmd + T`         | Cycle themes                   |
| `Ctrl/Cmd + F`         | Search in current document     |
| `Ctrl/Cmd + Shift + F` | Search in folder or focus mode |
| `Ctrl/Cmd + Plus`      | Zoom in                        |
| `Ctrl/Cmd + Minus`     | Zoom out                       |
| `Ctrl/Cmd + 0`         | Reset zoom                     |
| `Ctrl + [`             | Toggle table of contents       |
| `Ctrl + \`             | Toggle file browser            |
| `Ctrl + Tab`           | Next tab                       |
| `Ctrl + Shift + Tab`   | Previous tab                   |
| `Space`                | Scroll down one screen         |
| `Shift + Space`        | Scroll up one screen           |
| `Escape`               | Close search or overlay        |

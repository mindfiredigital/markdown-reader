Everything a dedicated Markdown reader needs

Built for people who read READMEs, specs, notes, docs, changelogs, and long Markdown files every day.

Reader-first layout

Open Markdown like a document, with centered typography, a calm reading surface, and no editor clutter.

Full Markdown rendering

GFM tables, task lists, code blocks, local images, Mermaid diagrams, KaTeX math, and callout blocks.

Search that feels native

Find text inside the current file, move between matches, and expand to folder-wide search when a folder is open.

Folders and tabs

Browse Markdown files from a folder, open multiple documents in tabs, and keep recent files close.

Live reload

When another editor changes the file, Markdown Reader refreshes the view without losing your scroll position.

Comfortable themes

Use GitHub Light, GitHub Dark, Notion, Nord, Minimal, or Dracula with zoom and reading width controls.

Why Markdown Reader ?

Built for reading, not editing

VS Code is an editor. GitHub is a website. Markdown Reader is a dedicated desktop viewer that opens .md files like finished documents.

Fast by default

Designed around quick launch, quick file open, and smooth reading for everyday Markdown documents, READMEs, and notes.

Private and offline

Your files stay local. No accounts, no cloud sync, no telemetry, and no network calls during normal reading.

Keyboard friendly

Open files, switch themes, search, zoom, toggle panels, and move through tabs with native shortcuts.
